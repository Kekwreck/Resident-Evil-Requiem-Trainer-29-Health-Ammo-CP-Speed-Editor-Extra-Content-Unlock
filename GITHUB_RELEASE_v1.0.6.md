## 🆕 5 NEW FEATURES in v1.0.6!

### 1. 🎚️ **Opacity Control** (F7/F8)
- **F7** = Decrease transparency (more see-through)
- **F8** = Increase opacity (more solid)
- Range: 20% to 100%

### 2. 📌 **Always On Top Toggle** (F9)
- **F9** = Toggle window always-on-top
- Window can now go behind game if needed

### 3. 💾 **Profile System** (F10/F11/Ctrl+1-3)
- **F10** = Save current profile
- **F11** = Load saved profile
- **Ctrl+1/2/3** = Switch between 3 profiles
- Saves all cheat states, opacity, settings!

### 4. ⏱️ **Cheat Timer** (F12)
- **F12** = Start/Stop 60-second countdown
- Auto-disables ALL cheats when timer ends
- Perfect for timed challenges!

### 5. 📥 **Minimize to Tray** (Ctrl+M)
- **Ctrl+M** = Hide to system tray
- Double-click tray icon = Restore
- Right-click = Menu (Restore/Exit)
